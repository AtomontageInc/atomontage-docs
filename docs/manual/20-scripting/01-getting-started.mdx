## Lua

Lua is the scripting language used in Atomontage. It is a powerful, efficient scripting language that is used for scripting game logic, ui and also the editor.

Lua is easy to learn! A good resource to learn Lua is [Learn Lua in 5 minutes](https://gist.github.com/roalcantara/49de782ab32385db78357192e6750c8e)
