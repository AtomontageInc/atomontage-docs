### number transformFingerprint {#number-transformFingerprint}
64-bit value that changes when the object's global (world) position, rotation, or scale change — including motion caused by an ancestor moving. Useful for cheap change detection.

### string name {#string-name}

### boolean active {#boolean-active}
Set the object to be active or inactive. Inactive objects are not updated or rendered. 
All its children also become inactive.

### boolean activeInHierarchy {#boolean-activeInHierarchy}
Readonly. Check if the object is active in the scene. It may be inactive because a parent is inactive.

### boolean save {#boolean-save}
Save this object in the hierarchy. If not saved it will be deleted after lua reset or server restart

### Transform transform {#Transform-transform}
Get the transform to modify the position, rotation and scale of the object

### boolean isDestroyed {#boolean-isDestroyed}
True if the object was destroyed. Note that references to this object will still be valid 

### string id {#string-id}

### integer subtreeVersion {#integer-subtreeVersion}
Monotonic counter bumped on this object and all of its ancestors whenever a child is added, removed, or reordered anywhere in its subtree (parent/unparent/reorder). Does NOT change for transform edits (use [`transformFingerprint`](#number-transformFingerprint)) or component add/remove. Cheap O(1) check for invalidating caches derived from the child hierarchy, e.g. `WithChildren()`.
